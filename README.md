# Marcus Soares — Portfólio

Site de portfólio pessoal. Arquivo único (`index.html`) — sem build, sem dependências locais.

---

## Sobre

Desenvolvedor Full-Stack com trajetória desde 2012: automações de navegador com TypeScript e Tampermonkey, especialização em Lua e JavaScript para modificações de jogos (FiveM — **Compton City**, 2019), e a partir de 2022 foco em backend Python, análise de dados, bots Discord e desenvolvimento web completo.

Hoje sou o único desenvolvedor do **BSL** (bsl.lat) — plataforma esportiva em produção com usuários reais, rodando em VPS desde 2023.
